# Calculator Project

A **web-based calculator** built with **HTML, CSS, and JavaScript**, designed to mimic real mobile calculator behavior including percentage (`%`) operations. Users can perform addition, subtraction, multiplication, division, and percentages in a clean and interactive interface.

---

## Features

- **Basic arithmetic:** Addition (+), Subtraction (-), Multiplication (×), Division (÷)
- **Percentage support:** Works like mobile calculators  
  Example: `450 - 20%` → `360` (calculated on pressing `=`)
- **AC button:** Clears all input
- **DEL button:** Deletes last character
- **Real-time input display:** Shows the expression as typed
- **Clean UI:** Simple and user-friendly interface

---

## How to Use

1. Open `index.html` in a web browser.
2. Click the buttons to enter numbers and operators.
3. Press `%` to append percentage (calculated when `=` is pressed).
4. Press `=` to calculate and display the result.
5. Use `AC` to clear and `DEL` to remove the last character.
